# screenstack

Stack video screenshots into one tall image — for turning mpv stills into memes.

Drop frames in, they sort chronologically by filename, hit **export png**. Hover a frame to reorder, duplicate, or remove it. Everything runs in the browser; nothing is uploaded anywhere.

**Use it:** [meowteusz.github.io/screenstack](https://meowteusz.github.io/screenstack/), or just open `index.html`.
